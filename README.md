_node-poe-api_ is a minimal module exposing the Path of Exile stash tabs api.

## Installation

Install `node-poe-api` and save to your `package.json` in one easy step.

With npm:

```
npm install node-poe-api --save
```

With yarn:

```
yarn add node-poe-api
```
